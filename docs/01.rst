.. raw:: html

  <h2 id="test111">1 总则</h2>

.. raw:: html

 <p style="text-align:justify"><a href="#id1.0.1"> 1.0.1</a> <span id="id1.0.1">为规范公路桥涵设计，按照安全、耐久、适用、环保、经济和美观的原则，制定本规范。</span></p>
 <p style="text-align:justify"><a href="#id1.0.2"> 1.0.2</a> <span id="id1.0.2">本规范适用于新建和改建各等级公路桥涵的设计。</span></p>
 <p style="text-align:justify"><a href="#id1.0.3"> 1.0.3</a> <span id="id1.0.3">公路桥涵结构的设计基准期为100年。</span></p>

 <p style="text-align:justify"><a href="#id1.0.6"> 1.0.6</a> <span id="id1.0.6">公路桥涵应进行抗风、抗震、抗撞等减灾防灾设计。</span></p>
 <p style="text-align:justify"><a href="#id1.0.7"> 1.0.7</a> <span id="id1.0.7">公路桥涵设计应满足环境保护和资源节约的有关要求。</span></p>
 <p style="text-align:justify"><a href="#id1.0.8"> 1.0.8</a> <span id="id1.0.8">公路桥涵设计除应符合本规范的规定外，尚应符合国家和行业现行有关标准的规定。</span></p>
